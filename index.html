INSTALL TROUBLESHOOTING

Follow these steps if your browser or Windows blocks the game client during download or launch.

01 // BROWSER DOWNLOAD WARNING

If your browser flags the game as an uncommon download, open the download item options, choose Keep, then confirm with Keep Anyway if needed.

02 // WINDOWS SMARTSCREEN OVERRIDE

If Windows shows a SmartScreen warning, click More Info and then choose Run Anyway to continue launching the game.

03 // RUN AS ADMINISTRATOR

Launching the installer as an administrator can help the game set up save files, settings, and local content correctly.
